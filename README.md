# tw-android-pm25
##功能
可以输入不同的城市名称来显示不同城市的 PM 值，默认城市为 西安。
##UI 说明
![main-page](https://raw.githubusercontent.com/nancymi/picture-transfer/master/pm25/main-page.png)

右上角：
* 刷新键：重新获取数据
* 区域键入键：写入城市名称重新获取数据

中间偏上：
* 左键：查看上一个 position 的 PM2.5 值
* 右键：查看下一个 position 的 PM2.5 值
